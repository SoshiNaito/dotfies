# Create PR Skill

以下の手順でプルリクエストを作成します。

## 手順

1. 最終チェックを実行:
   - 全テストがパス
   - Lintチェック
   - セキュリティスキャン

2. 変更内容に基づいてコミットメッセージを生成

3. 変更をステージングしてコミット

4. リモートブランチにプッシュ

5. 以下を含むPRを作成:
   - 変更内容の要約
   - 関連Issue
   - 実施したテスト
   - スクリーンショット（該当する場合）

PRタイトル: $ARGUMENTS

今すぐワークフローを実行してください。

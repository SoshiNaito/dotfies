# コミット前フック

## 実行するチェック

### 1. Lintチェック
プロジェクトのLintツールを実行します。

### 2. 型チェック（型付き言語の場合）
静的型解析を実行します。

### 3. テスト実行
関連するテストを実行します。

### 4. セキュリティスキャン
依存関係の脆弱性をチェックします。

## レポート形式
```
✅ Lint: 合格
✅ 型チェック: 合格
✅ テスト: XX件合格、0件失敗
⚠️  セキュリティ: [警告があれば表示]
```

いずれかのチェックが失敗した場合、コミットを停止してユーザーに報告します。

## プロジェクトごとの設定

各プロジェクトで使用するコマンドを `.claude/CLAUDE.md` に記載してください:

```markdown
## コマンド
- Lint: `[プロジェクトのlintコマンド]`
- 型チェック: `[プロジェクトの型チェックコマンド]`
- テスト: `[プロジェクトのテストコマンド]`
- セキュリティ: `[プロジェクトのセキュリティスキャンコマンド]`
```

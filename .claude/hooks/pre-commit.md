# コミット前フック

## 実行するチェック

### 1. Lintチェック
```bash
npm run lint
```
失敗した場合は停止し、ユーザーに修正を依頼します。

### 2. 型チェック
```bash
npm run type-check
```

### 3. テスト実行
```bash
npm test
```

### 4. セキュリティスキャン
```bash
npm audit
```

## レポート形式
✅ Lint: 合格
✅ 型チェック: 合格
✅ テスト: 45件合格、0件失敗
⚠️  セキュリティ: 中程度の脆弱性2件発見

いずれかのチェックが失敗した場合、コミットを停止してユーザーに報告します。
